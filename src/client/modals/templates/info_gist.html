How to set up a CLA in Gist

What is GitHub Gist?

Gist is a simple way to share text files, snippets and pastes with others. All Gists are Git repositories, so they are automatically versioned, forkable and usable from Git.

How can I create a Gist file?

  • Go to gist.github.com and sign in with your GitHub account
  • Enter a prefered file name and content of your contributor license agreement
  • Choose whether you want to keep it private or make it public (for more information about this, see: github documentation ).

How can I use Gist for CLA assistant?

All you need to do is to copy and paste (or drag and drop) the URL of your Gist into CLA assistant, add a repository and link both.

What happens if I edit the Gist file?

CLA assistant always shows you the current version of your Gist file. Users who accept your contributor license agreement, sign only the current version. If you change the content of your CLA, each contributor has to accept the new version of the CLA for new pull requests.
