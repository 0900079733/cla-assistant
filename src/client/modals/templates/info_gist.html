GitHub Gist

What is Gist?

Gist is a simple way to share snippets and pastes with others. All gists are Git repositories, so they are automatically versioned, forkable and usable from Git.

How can I create a Gist?
  • Go to gist.github.com and sign in with your GitHub account
  • Enter a prefered file name and content of your contributor license agreement
  • Choose whether you want to keep it private or make it public (for more information about differnce between private and public files see github documentation )
How can I use Gist for CLA assistant?
All you need to do is just copy and paste (or drag and drop) the url of your Gist into CLA assistant and push the update button
What happens if I edit the Gist file?
CLA assistant shows allways the current version of your Gist file. Users who accept your contributor license agreement, do it for the current version. If you change the content of your CLA later each contributor has to accept the new version of the CLA even if they have accepted the previous version. But they have to do it only for new contributions / pull requests.
